:::info Authentication Required
To use this playground, add your Conviso API key to the HTTP HEADERS section:

1. Click on **HTTP HEADERS** at the bottom left of the playground panel.
2. Replace `VALUE_HERE` with your actual API key.

:::

Once the header is set, you can execute your queries.
